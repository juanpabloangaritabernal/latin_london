InTrO

Administrador

Actualiza la línea de tiempo cargando un Excel. Flujo recomendado:

  1. Descarga la plantilla con los datos actuales.
  2. Edita o agrega filas en Excel (una fila = un evento).
  3. Carga el archivo: el sistema valida, previsualiza y aplica.
  4. Exporta el JSON y pídele a Claude que lo integre al HTML para hacerlo permanente.

Columnas obligatorias: id, series, date_label, sort_date, category, title_en, title_es, title_pt, text_en, text_es, text_pt, image

Imágenes: sube los JPG a la raíz de tu repositorio y usa solo el nombre del archivo (ej. A3_TimeMusic-04.jpg), o pega una URL completa; las URLs de GitHub tipo blob se convierten automáticamente en directas.
